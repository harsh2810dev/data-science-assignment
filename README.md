# International T20 Cricket Data Analysis

## Project Overview

This project performs exploratory data analysis on International T20 cricket match data using Python data science libraries.
The goal of the project is to understand the dataset, perform data cleaning, apply statistical analysis, and generate visualizations to identify patterns and insights in T20 cricket matches.

The analysis was implemented using a notebook environment in Google Colab.

## Objectives

* Load and explore the dataset
* Perform data cleaning and preprocessing
* Apply functions from Pandas and NumPy
* Perform statistical analysis
* Generate visualizations using Matplotlib and Seaborn
* Interpret trends and insights from the dataset

## Dataset

The dataset used in this project:

International_T20_Data.csv

The dataset contains information related to international T20 matches, including:

* Match venues
* Cities
* Match dates
* Winning teams
* Run margins
* Wicket margins
* Toss decisions
* Match outcomes

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Google Colab

## Data Analysis Process

### Data Loading

The dataset is loaded using the Pandas library.

### Data Exploration

Initial exploration includes:

* Viewing the first and last rows of the dataset
* Checking dataset dimensions
* Inspecting column names and data types
* Generating summary statistics

### Data Cleaning

The cleaning process includes:

* Identifying and handling missing values
* Removing duplicate records
* Renaming columns where necessary
* Creating new derived columns

### Statistical Analysis

NumPy functions are used to calculate:

* Mean
* Median
* Standard deviation
* Variance
* Minimum and maximum values

### Data Aggregation

Pandas operations used in the project include:

* groupby
* value_counts
* pivot_table
* sorting and filtering operations

### Data Visualization

The following visualizations were created:

* Bar charts
* Histograms
* Pie charts
* Count plots
* Box plots
* Correlation heatmaps

These visualizations help identify patterns in match results, team performance, and match conditions.

## Conclusion

This project demonstrates how Python data science libraries can be used to perform exploratory data analysis and visualization on sports datasets. The analysis provides insights into match outcomes, team performance, and other important patterns in international T20 cricket matches.
